# Techium Homepage 🌐

A clean and responsive **homepage project** built with **HTML5 & CSS3**, following modern CSS best practices and Holberton-style requirements.

---

## 📖 Overview

This project is part of a front-end learning path focused on mastering **CSS fundamentals and advanced concepts**. The goal is to understand how styling works under the hood, how browsers interpret CSS, and how to build consistent layouts without relying on frameworks.

The project uses a provided starter HTML file and applies structured, maintainable CSS to achieve the final design.

---

## 🎯 Learning Objectives

By completing this project, you should be able to explain:

* CSS **selectors, properties, and values**
* The difference between **block** and **inline** elements
* How to ensure **cross-browser consistency** using CSS reset
* How to define and use **CSS variables**
* Differences between **inline, embedded, and external CSS**
* How **grid systems** work using floats
* Icons: **webfonts vs SVG icons**
* **Pseudo-classes** vs **pseudo-elements**
* Creating **background gradients**
* Animating elements with **CSS animations**
* Applying **2D and 3D transforms**
* What **vendor prefixes** are and why they exist

## ⚙️ Requirements

* Allowed editors:

  * `vi`, `vim`, `emacs`
  * `VSCode`, `Atom`
* All files must:

  * End with a **new line**
  * Start with a **comment describing the task**
* A `README.md` file is mandatory
* Code must:

  * Be **W3C compliant**
  * Pass **W3C Validator** checks
* No external CSS frameworks

---

## ▶️ How to Run

1. Clone the repository
2. Open `index.html` in **Google Chrome**
3. Make sure the CSS file is linked correctly:

```html
<link rel="stylesheet" href="styles/main.css">
```
