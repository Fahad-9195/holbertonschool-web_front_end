# Advanced HTML

This project is part of the **Holberton School Web Front-End** curriculum and focuses on **advanced HTML concepts** and best practices used in modern web development.

---

## 🎯 Learning Objectives

By the end of this project, you should be able to:

* Use semantic HTML elements correctly
* Structure complex web pages using clean and readable HTML
* Improve accessibility (a11y) using proper tags and attributes
* Embed media elements such as images, audio, and video
* Work with forms and advanced input types
* Follow HTML best practices and standards

---

## 🛠️ Technologies Used

* **HTML5**
* **W3C HTML Validator**
* Web Browser (Chrome, Firefox, Edge)

## ▶️ How to Run

1. Clone the main repository:

   ```bash
   git clone https://github.com/USERNAME/holbertonschool-web_front_end.git
   ```
2. Navigate to the Advanced HTML directory:

   ```bash
   cd holbertonschool-web_front_end/advanced_html
   ```
3. Open `index.html` in your browser

---

## ✅ Requirements

* All HTML files should end with a new line
* No external libraries unless explicitly allowed
* Code must pass **W3C validation**
* Use semantic HTML tags where appropriate

---

## 🧪 Validation

To validate your HTML files:

* Use the **W3C Markup Validation Service**
* Fix all errors and warnings before submission
