# Holberton School – Web Front End

This repository contains **Front-End Web Development** projects and exercises as part of the **Holberton School** curriculum. It focuses on building modern, responsive, and user-friendly web interfaces using core front-end technologies.

---

## 🎯 Learning Objectives

* Understand the structure of web pages using **HTML5**
* Style and layout interfaces using **CSS3**
* Apply **JavaScript** for interactivity and dynamic behavior
* Build **responsive designs** for different screen sizes
* Follow best practices and write clean, maintainable code

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3** (Flexbox, Grid)
* **JavaScript (ES6+)**
* **Git & GitHub**
* Browser Developer Tools

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/USERNAME/holbertonschool-web_front_end.git
   ```
2. Navigate to the desired project folder
3. Open `index.html` in your web browser

## ✅ Requirements

* A modern web browser (Chrome, Firefox, Edge)
* Code editor such as **VS Code**
* Basic knowledge of the terminal
